## Check Odd or Even

This program finds the number is **Odd or Even**.

### Java Implementation

```java
    public static void checkOddEven(int number) {
        if (number % 2 != 0) {
            System.out.printf("%d is an Odd Number", number);
        }
        else {
            System.out.printf("%d is an Even Number", number);
        }
    }
```
